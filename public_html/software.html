Dataset

  • GenderLex : Lexical resource for examining gender bias beyond occupational titles (English & Japanese).
  • est-gender-bias-dataset : Estonian gender bias evaluation set for measuring bias in a genderless language.
  • Textual-Image-Caption-Dataset : COCO based textual visual context dataset for image captioning and semantic relatedness.
  • Gulf-Arabic-Tweets-2018-2020 : Cleaned Gulf Arabic dialect tweets (2018 to 2020) for text classification.
  • Japanese-wiki-dump-sentence-dataset : ~5.1M clean Japanese full sentences with context, for unsupervised semantic similarity.
